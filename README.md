# Manufacturing-Analytics
A Mathematical Analytics project uses mathematical models, statistics, and data analysis techniques to study patterns and trends in data. The goal is to make predictions, solve real-world problems, and support better decision-making using numerical methods and visualizations.
The project analyzes manufacturing performance using multiple BI tools.
 Dashboards created in Excel, Power BI, Tableau and supported with MySQL database.
 Focus on manufacturing quantity, processed quantity, rejected quantity, and wastage.
 Comparative visual analysis across employees, machines, and departments
